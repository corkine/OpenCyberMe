# ICE Manager

ICE 特性（文档和评审）管理器，使用 clojure 和 clojurescript 搭建。

## Prerequisites

使用 [Leiningen][1] 构建，依赖 postgreSQL 数据库。

[1]: https://github.com/technomancy/leiningen

## Running
    
    //backend
    lein repl

    //frontend
    lein shadow watch app
